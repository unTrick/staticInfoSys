VPN Issues Setting up Pulse Secure
1. The Pulse Secure application will be automatically installed on your company-provided computer
2. Launch the application
3. In the dialog box that opens, click + to add a new connection
4. For users in North America, enter the following Name and Server URL

 • Name: MVW VPN
 • Server URL: vpn.mvwc.com

5. Click Connect
6. Choose the Corporate Users realm
7. When prompted, enter your EID and password (for legacy MVW associates) or your network credentials (for legacy ILG associates)
8. You will be prompted to enter the Security Code or authenticate via Thumbprint or Facial Recognition via the PingID mobile app. Open the app on your smartphone to authenticate
9. Click Connect
