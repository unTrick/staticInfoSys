Account Issues Password Reset
1. Look up the user’s EID on Active Directory
2. Right click the user’s account and then select, “Reset Password”
3. Make sure to check those two boxes for user to change the password to their desired password
4. Hit “OK” and changes should be applied real time. Have the user now sign in to www.mymvwc.com and use the temporary password for them to set the desired password
Note

Password must be at least 8 in length. Have at least 1 upper case, lower case and a number. Special characters are not required but recommended.
