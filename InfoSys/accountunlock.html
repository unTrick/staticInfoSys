Account Issues Unlock Account
1. Look up the user’s EID on Active Directory
2. Open user’s Properties and go to ‘Account Tab’
As you can see, user is locked out
3. Check the Check box to unlock the account and then hit Apply and Ok as shown in the images below
4. Have the user wait for at least 5 seconds for the changes to take in effect
Note

This will only work if the application user’s trying to access requires a SSO login.
