Please perform these steps in order to avert the error 47’s in CNAH005, until such a time that the developer can fix the issue.

When you’re at the Property prompt - Press the function key
This brings up a pop-up window where you can find the account by name, contract number or member number if you have it
I chose option 2, then entered the account #
The next screen brings up all the owners on the contract - simply press on one of the names
It will populate the upper part of the screen and bring up the same pop-up you had before where you select all the lines of a similar owner
BUT – this time when you press function key, you won’t get an error!!!
